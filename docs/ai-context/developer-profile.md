# Developer Profile / Perfil do Desenvolvedor

## EN

### Person and context

- Founder-developer: Carlos Siqueira.
- Parent building a focused educational product for his own child first.
- Works across multiple machines (Windows laptop, Windows desktop, Mac mini).

### Working style

- Solo shipping with pragmatic prioritization.
- Fast iteration with AI assistance.
- Strong bias toward production readiness and store approval.

### Tooling context

- Uses Cursor as primary environment.
- May interact with other AI assistants and editors.
- Needs portable project memory independent of local machine state.
- Android device QA: Windows (USB + `flutter run`). iOS / physical iPhone: requires Mac (Xcode, signing), usually the Mac mini when available—not from Windows only.

### Collaboration preferences for AI

- Keep responses objective and action-oriented.
- Prioritize child safety, compliance, and release risk reduction.
- Prefer clear next steps over generic advice.
- **Language:** for any Portuguese text aimed at the team or product (and for assistant replies when the user uses Portuguese), use **Brazilian Portuguese (pt-BR)** only—not European Portuguese (pt-PT). Prefer vocabulary and phrasing from Brazil (e.g. *celular*, *tela*, *baixar*; avoid *telemóvel*, *ecrã*, *tu* as default).
